**Title First Addon** reorders question listings in Q2A so that the **question title appears before the author/meta information**. Designed specifically for the **MayroPro-master theme**, but also works with other themes that don’t fully override `q_item_main()`.

---

## Features

- Display question title before author in archive/list pages  
- Safe and fully compatible with Q2A 1.9+  
- Admin toggle to enable or disable the feature  
- Works with MayroPro-master theme and most other themes  
- Easy to install and activate, no core files modified
