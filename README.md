# Title First Addon for MayroPro-master

[![Version](https://img.shields.io/badge/version-1.0-blue)](https://github.com/knowledgebuzz/title-first-addon-mayropro)  
[![License](https://img.shields.io/badge/license-GPLv2-red)](LICENSE)  
[![Donate](https://img.shields.io/badge/Donate-❤️-red)](https://unitedafrica.digital/donate/)

---

**Title First Addon** reorders question listings in Q2A so that the **question title appears before the author/meta information**. Designed specifically for the **MayroPro-master theme**, but also works with other themes that don’t fully override `q_item_main()`.

---

## Features

- Display question title before author in archive/list pages  
- Safe and fully compatible with Q2A 1.9+  
- Admin toggle to enable or disable the feature  
- Works with MayroPro-master theme and most other themes  
- Easy to install and activate, no core files modified  

---

## Installation

1. Download or clone this repository:  
```bash
git clone https://github.com/knowledgebuzz/title-first-addon-mayropro.git
```

2. Upload the `qa-plugin` folder to your Q2A installation's `/qa-plugin/` directory  

3. Activate the plugin in **Admin → Plugins**  

4. Optional: Go to **Admin → Plugins → Title First Addon Info** to toggle the feature
